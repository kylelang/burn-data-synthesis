In this script, we’ll explore/clean the burn data and prepare them for synthesis/imputation. We’ll do the synthesis/imputation in a separate script.

Data Ingest

First, we’ll clear the workspace, load necessary packages, and define some global variables.

rm(list = ls(all = TRUE))

library(ggplot2)
library(plotly)
library(viridis)
library(haven)
library(mice)
library(xtable)
library(knitr)

dataDir <- "../../data/"
fn      <- "Social Impact Simulatie Totaal.sav"

dat0 <- read_sav(paste0(dataDir, fn))

What kind of non-scale data do we have for the cases without patient scale data?

  • None of these cases have data on the relation_t3 variable or the rel_duration_t3 variable.
  • As seen in the tables above, 10 cases have observed partner demographics.
  • The response rates for the non-partner, non-scale variables looks pretty good, but these 41 cases still face a brutal imputation challange.
    • For the 31 cases with missing partner demographics, we’re talking about imputing 293 or 294 variables from 11 or 12 predictors.
    • For the 10 cases with observed partner demographics, the story isn’t much nicer. We still need to impute 289 or 290 variables from 12 or 16 predictors.

If we have to exclude the 41 cases with completely missing patient scale data, we’ll still have 225 cases to work with. However, if we also have to exclude the cases with completely missing partner data, we’ll only have 110 cases.
